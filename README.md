# 🌍 Travel Booking Platform

Full-stack travel booking app with **Black & White Techie UI/UX**. Search flights, hotels, and get best deals instantly!

## ✨ Features

- 🔍 **Smart Search** - Flights, Hotels, Attractions
- 💰 **Price Comparison** - Best deals across platforms
- 🗺️ **Google Maps Integration** - Route planning
- ✈️ **Flight Booking** - Multi-city options
- 🏨 **Hotel Management** - Live availability
- 🎯 **Itinerary Builder** - Custom trip planning

## 🛠️ Tech Stack

**Frontend:** React 18 + TypeScript + Tailwind CSS + Redux
**Backend:** Node.js + Express + MongoDB + Redis
**APIs:** Google Maps, Amadeus, Booking.com, Stripe

## 🚀 Quick Start

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend
cd frontend && npm install && npm run dev
```

**Access:** http://localhost:5173

# 🚀 Setup Guide\n\n## Quick Start\n\n### 1. Clone Repository\n```bash\ngit clone https://github.com/ajay2025rathore-source/travel-booking-platform.git\ncd travel-booking-platform\n```\n\n### 2. Backend Setup\n```bash\ncd backend\nnpm install\ncp .env.example .env\nnpm run dev\n```\n\n### 3. Frontend Setup (New Terminal)\n```bash\ncd frontend\nnpm install\nnpm run dev\n```\n\n### 4. Access Application\n- Frontend: http://localhost:5173\n- Backend: http://localhost:5000\n\n## Docker Setup\n```bash\ndocker-compose up -d\n```\n\n## Environment Variables\n\n### Frontend (.env)\n```\nVITE_API_URL=http://localhost:5000\nVITE_GOOGLE_MAPS_API_KEY=your_key\n```\n\n### Backend (.env)\n```\nNODE_ENV=development\nPORT=5000\nMONGODB_URI=mongodb://localhost:27017/travel_db\nREDIS_URL=redis://localhost:6379\nJWT_SECRET=your_secret\nGOOGLE_MAPS_API_KEY=your_key\n```\n\n## Next Steps\n1.
